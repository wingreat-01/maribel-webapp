⚙ Google Apps Script Backend Configuration

Backend is managed via Google Apps Script (Code.gs from the repository).

  1. Open your Google Sheet → Extensions → Apps Script
  2. Copy the contents of Code.gs from the GitHub repository and paste it there
  3. Set SHEET_ID at the top of Code.gs to your Google Sheet ID
  4. Click Deploy → New deployment → Web app
  5. Set Execute as = Me  |  Who has access = Anyone
  6. Copy the Web App URL and paste it below

⚠ If prompted for permissions, click Review permissions → Allow.
